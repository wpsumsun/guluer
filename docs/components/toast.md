<script>
export default {
  mounted() {
    import('../../src/toast').then(module => {
      // use code
    })
  },
  methods: {
    showToast(position) {
      this.$toast('独钓寒江雪', {
        position
      })
    }
  }
}
</script>
| 参数        | 说明           | 类型  |  可选值  | 默认值 |
| ------------- |:-------------:| -----:|-----:|-----:|
| closeButton      | 关闭按钮设置 | Object |   | `{text: '关闭',callback(toast) {toast.close()}}`|
| enableHtml      | 是否支持html      |   Boolean |   |  false |
| position | 出现位置      |    String |  `top, middle, bottom` | top |
